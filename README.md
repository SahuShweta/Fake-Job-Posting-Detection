# 🛡️ Fake Job Posting Detection

## 📌 Project Overview

Fake Job Posting Detection is a Machine Learning based web application that predicts whether a job posting is **Real** or **Fake**.

The project uses Natural Language Processing (NLP) techniques with TF-IDF Vectorization and a Machine Learning classification model.

Users simply enter the job information, and the model predicts whether the posting is fraudulent.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/SahuShweta/Fake-Job-Posting-Detection.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python app.py
```
